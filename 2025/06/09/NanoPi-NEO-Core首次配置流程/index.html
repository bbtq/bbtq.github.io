NanoPi NEO Core首次配置流程

发表于2025-06-09|更新于2025-06-10
|浏览量:

原本的配置流程链接

准备工作

  1. NanoPi NEO Core 主板
  2. 8G 及以上的 TF 卡和读卡器（TF卡推荐查看官方WIki推荐购买）
  3. 一个microUSB接口的外接电源（电源线）
  4. USB 转串口
  5. 水下机器人主板（提供网口和供电）
  6. 软件

    安装SSH软件，如 FinalShell 、MobaXterm 等等。（这里推荐Finalshell）
    安装串口终端，如 Putty 、MobaXterm等等。（这里推荐Putty）
    安装系统烧写工具，win32diskimager、Rufus 等等。（这里推荐Rufus，微软应用商店也可以直接下载）
    准备系统镜像，官方 Wiki 下载 镜像 ，这里选择official images/SD-to-eMMC images下最新的 h3-eflasher-friendlycore-focal-4.14-armhf-20250402.img.gz 。

具体流程

  1. 插上TF卡后，使用 Rufus 烧录镜像
  2. 烧录完成，拔出。将TF卡插入NanoPi NEO Core主板上的卡槽内。
  3. 使用USB转串口工具，连接到UART0，同时共地
  4. 打开putty，选择串口（根据系统的设备管理器查看端口），波特率115200 ，连接。
  5. 烧写到eMMC
    1. 确认一切准备完成后上电启动。可以看到STAT灯（黄）闪烁，这是已经成功启动系统。
    2. 可以看到putty终端中不断打印系统信息，最后登录系统。

      普通用户：
      用户名：pi
      密码：pi
      Root用户：
      用户名：root
      密码：fa

    3. 登录系统后，使用如下命令进行烧写
    4. 出现 Installation Complete 等信息后，poweroff。
    5. 断电后，拔出TF卡
  6. 修改为静态ip (vim的相关操作自行百度，i:插入/编辑，wq:保存退出)
